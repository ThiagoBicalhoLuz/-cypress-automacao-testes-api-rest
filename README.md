# 🧪 Cypress - Automação de Testes em API REST

## 🎯 Sobre o Projeto
Este repositório foi criado com o objetivo de centralizar meus estudos e práticas de automação de testes voltados para Back-end (APIs RESTful). A ideia é evoluir na criação de testes automatizados robustos, rápidos e confiáveis utilizando o ecossistema do Cypress.

## 🛠️ Tecnologias Utilizadas
* **[Cypress](https://www.cypress.io/)**: Framework principal para a automação dos testes.
* **JavaScript**: Linguagem base para a escrita dos scripts.
* **Node.js**: Ambiente de execução.
